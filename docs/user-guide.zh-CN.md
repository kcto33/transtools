# ScreenTranslator 使用指南（中文）

## 环境要求
- Windows 11 22H2+（build 22621+）
- 从源码运行需安装 .NET 8 SDK

## 启动与托盘菜单
- 应用启动后常驻系统托盘。
- 双击托盘图标：开始选区翻译。
- 右键托盘图标菜单：
  - 开始翻译
  - 历史粘贴
  - 截图贴图
  - 设置
  - 开机启动
  - 退出

## 选区翻译
1. 按 `Ctrl+Alt+T`，或在托盘菜单点击“开始翻译”。
2. 鼠标拖拽选择区域，按 `Esc` 可取消。
3. 选区上方会出现翻译气泡。

气泡交互：
- 左键：在“译文 / 原文”之间切换。
- 右键：复制当前显示文本。
- 点击外部或约 6 秒后自动关闭。

## 历史粘贴（剪贴板选择器）
1. 按 `Ctrl+Shift+V`，或在托盘菜单点击“历史粘贴”。
2. 光标附近弹出最近文本列表。
3. `↑/↓` 选择，`Enter` 粘贴，`Esc` 关闭。

说明：
- 仅保存文本。
- 自动去重（最新条目置顶）。
- 历史数量可在设置中调整（1-20）。

## 截图与贴图
1. 按 `Ctrl+Alt+S`，或在托盘菜单点击“截图贴图”。
2. 拖拽框选矩形区域。
3. 工具栏操作：
   - 贴图：固定到屏幕
   - 复制：复制到剪贴板
   - 保存：保存到文件
   - 长截图：进入长截图会话
   - 自由截图：切换到自由形状截图
   - 取消：关闭截图层

矩形模式快捷操作：
- `Enter`：复制选区
- `Esc`：取消
- 鼠标中键：直接贴图

自由截图模式：
- 按住左键绘制路径，松开完成。
- 工具栏支持：贴图 / 复制 / 保存 / 重画 / 取消。

## 长截图（手动滚动拼接）
1. 按 `Ctrl+Alt+S` 进入截图模式。
2. 先框选可滚动内容区域。
3. 点击工具栏“长截图”。
4. 会话立即开始：
   - 选框变为红框并固定在屏幕位置。
   - 选框附近出现控制条。
   - 选框旁出现预览窗（当前帧 + 拼接尾部 + 匹配标记）。
5. 采集过程中：
   - 在目标应用里手动滚动（滚轮/滚动条/PageDown/Space）。
   - `暂停`：进入蓝框可调状态，可拖动/缩放选框。
   - `继续`：恢复红框锁定并继续采集。
   - `停止`：输出当前拼接结果。
   - `取消`：直接结束，不输出结果。
   - `隐藏预览/显示预览`：切换预览窗。
6. 匹配失败处理：
   - 预览轨道出现红色标记。
   - 点击红标记或点击拼接预览图可手动修正接缝。
   - 也可点击 `跳过` 忽略当前失败帧。
7. 完成后：
   - 结果默认自动复制到剪贴板。
   - 若开启“截图后自动保存”，会自动落盘。
   - 仍可手动执行 `复制 / 保存 / 贴图`。

说明：
- 当前版本仅支持纵向拼接。
- 只有检测到手动滚动输入时才会进入下一帧处理。
- 到底检测、最大帧数、最大高度、连续失败保护都会触发自动停止。

## 设置说明
在托盘菜单打开“设置”。

通用页：
- 开机启动
- 界面语言（切换后需重启）
- 快捷键配置
- 历史粘贴数量
- 默认源语言 / 目标语言
- 截图设置：自动复制、自动保存、保存路径、文件名格式
- 长截图参数：
  - 每步滚轮刻度
  - 帧间隔（ms）
  - 最大帧数
  - 最大总高度（px）
  - 无变化阈值（%）
  - 连续无变化次数

API 页：
- Provider：`mock` / `youdao` / `deepl` / `google`
- 凭据：
  - Youdao：AppId + AppSecret
  - DeepL/Google：API Key
- 设置文件：`%AppData%\ScreenTranslator\settings.json`
  - 密钥使用 Windows DPAPI 保护

气泡样式页：
- 可分别设置“翻译气泡”和“历史粘贴气泡”的颜色、字体、内边距、最大宽度。
- 支持一键重置默认值。

## 常见问题
- 快捷键无效：通常是与其他应用冲突，改用其他组合键。
- 翻译失败：气泡会显示失败提示，请检查 Provider 配置和网络访问。
